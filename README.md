# React Coursera Capstone Project

This is Syed Ali Raza Jaffery capstone project for Coursera.

### Install Dependency:
Run Command:
```
npm install
```

### Run Project:
Run Command:
```
npm start
```

### Run Test:
Run Command:
```
npm run test
```

Test is in file called **Booking.test.js** in **src/components/Booking**
